# PaketJunge
GUI for building OSI layer oriented packets (Beta).

Requires WinPcap https://www.winpcap.org/

Current release: [Download](https://bit.ly/2mRoZVy)

Execute **PaketJunge.View.exe** to start.

Watch traffic using wireshark.
